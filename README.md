# A4 Verb Worksheet

日本語動詞活用のA4印刷用ワークシートです。

## 機能

- 教材を選択
- 問題だけを印刷
- 「答えを印刷する」にチェックすると解答欄へ答えを表示
- 画面上は約50%プレビュー
- 印刷時はA4縦の実寸

## MD形式

```md
title: ない形

01: 書きます
02: 書かない
03: 食べます
04: 食べない

05: 聞きます
answer05: 聞かない
```

`05`は問題欄、`answer05`は解答欄に対応します。

## 教材追加

新しいMDファイルを追加し、`files.json`にも登録します。

```json
{
  "file": "5jishokei.md",
  "label": "辞書形"
}
```
